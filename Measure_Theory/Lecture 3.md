# Borel $\sigma$-algebra

Let $\Omega$ be a topological space. Then the $\sigma$-[[Measure-Theory-Notes/Measure_Theory/Lecture 2#Definition| algebra generated]] by open sets in $\Omega$ is called the Borel $\sigma$-algebra for $\Omega$ and is denoted by $\mathcal{B}_{\Omega}$.

# $G_\delta$-sets

A set $E$ in $\Omega$ is called $G_\delta$ if it is an intersection of open sets.

# $F_\sigma$-sets

A set $E$ in $\Omega$ is called $F_\sigma$ if it is an union of closed sets.

## Proposition

Let $\mathcal{S} \subseteq \mathcal{P}(\Omega)$ be a [[Measure-Theory-Notes/Measure_Theory/Lecture 2#Definition (Semi-algebra)|semi algebra]]. Then for $A \in \mathcal{A}(\mathcal{S}) \ \exists$ collection of sets $E_1, E_2 ,\dots\in \mathcal{S}$ s.t.

$$A = \bigcup_{i=1}^k E_i$$

$E_i \cap E_j = \emptyset$ if $i \neq j$

### Proof
Let $$\mathcal{G} = \left\{ \bigcup_{i=1}^{n} E_i : E_i \in \mathcal{S}, 1 \leq i \leq n, E_j \cap E_k = \phi, k \neq j \right\}$$

It follows that $\mathcal{S} \subseteq \mathcal{G} \subseteq \mathcal{A}(\mathcal{S})$. It was enough to show that $\mathcal{G}$ is an algebra

Take $A, B \in \mathcal{G}$, Then $A = \bigcup_{i=1}^{n} E_i, B = \bigcup_{j=1}^{m} F_j, E_i \cap E_j = \phi$ if $i \neq j, F_k \cap F_l = \phi$ if $k \neq l$.

$$A \cap B = \bigcup_{i=1}^{n} \bigcup_{j=1}^{m} E_i \cap F_j \in \mathcal{G} \text{ (since it is disjoint union of sets.)}$$

$$(E_i \cap F_j) \cup (E_{i'} \cap F_{j'}) = \phi$$

Now $A^c = \bigcap_{i=1}^{n} E_i^c$ since $E_i \in \mathcal{S} \ \exists \ F_{i,1}, \dots, F_{i, m_i} \in \mathcal{S}$ s.t. $E_i^c = \bigcup_{j=1}^{m_i} F_{ij}$

Therefore:-

$$A^c = \bigcap_{i=1}^{n} \bigcup_{j=1}^{m_1} F_{ij} = \bigcup_{k_1=1}^{m_1} \dots \bigcup_{k_n=1}^{m_n} \underbrace{(F_{1k_1} \cap F_{2k_2} \cap \dots \cap F_{nk_n})}_{\text{disjoint unions}} \in \mathcal{S}$$

# Definition: (Measure)

Let $\Omega$ be a non empty set equipped with a $\sigma$-[[Measure-Theory-Notes/Measure_Theory/Lecture 2#Definition ($ sigma$-Algebra)|algebra]] $\mathcal{F}$ .A measure on $\mathcal{F}$ is function $\mu : \mathcal{F} \longrightarrow [0, \infty]$ st.

1. $\mu(\phi) = 0$
    
2. If $E_1, E_2, \dots$ is a disjoint sequence of sets in $\mathcal{F}$ then $$\mu(\bigcup_{j=1}^{\infty} E_j) = \sum_{j=1}^{\infty} \mu(E_j)$$
___
**Example:-**

The pair $(\Omega, \mathcal{F})$ is called a measurable space.

If $\mu$ is a measure $(\Omega, \mathcal{F})$ then $(\Omega, \mathcal{F}, \mu)$ is a measure space.
___

**Example:-** 
Let $\Omega$ be a countable set and $\mathcal{F} = \mathcal{P}(\Omega)$ and $\{ n_i \}$ is a sequence of +ve integers. 

Define $$\mu(A) = \sum_{j=1}^{\infty} n_j \chi_A(x_j)$$Check $\mu$ is a measure.
____
On $(\Omega, \mathcal{F})$ define $$\mu(A) = \begin{cases} \# A & \text{if } A \text{ is finite} \\ \infty & \text{if } A \text{ is infinite} \end{cases}$$
___

**Example**

$\Omega = (0, 1]$, $\mathcal{F} = \mathcal{F}(\{ (a, b] : 0 \leq a < b < 1 \} \cup \{ \phi \})$

Define $$\mu((a, b]) = \begin{cases} \infty & \text{if } a = 0 \\ b - a & \text{if } a > 0 \end{cases}$$

Prove that $\mu$ is finitely additive i.e. $$(a, b] = \bigcup_{j=1}^n (a_j, b_j]$$

then $$\mu((a, b]) = \sum_{j=1}^n \mu((a_j, b_j])$$

Now, $\mu((a, 1/2]) = \infty$ and $$(0, 1/2] = \bigcup_{j=1}^{\infty} (\frac{1}{2^{j+1}}, \frac{1}{2^j}]$$

But $$\sum_{j=1}^{\infty} \mu((\frac{1}{2^{j+1}}, \frac{1}{2^j}]) = \sum_{j=1}^{\infty} (\frac{1}{2^j} - \frac{1}{2^{j+1}})$$
