**Note:** $\mathcal{A} \subseteq \mathcal{P}(\Omega)$, $\mu : \mathcal{A} \to [0, \infty]$, which is a **pre-measure**.

i.e., if $\{A_n\} \subseteq \mathcal{A}$ disjoint then

$$\mu(\cup_n A_n) = \sum_n \mu(A_n)$$

---

$$\mu^* : \mathcal{P}(\Omega) \to [0, \infty]$$

$$\mu^*(A) = \inf \{ \sum_{n \geq 1} \mu(A_n) : \{A_n\} \text{ is cover of } A \text{ in } \mathcal{A} \}$$

---

$$M_{\mu} = \{ A \in \mathcal{P}(\Omega) : \mu^*(E) \geq \mu^*(E \cap A) + \mu^*(E \cap A^c), \forall E \subseteq \Omega \}$$

$M_{\mu}$ is a $\sigma$-algebra.

---

$\nu = \mu^*|_{M_{\mu}}$ is a complete measure space.

# Definition

A function $f: (\Omega, \mathcal{F}) \to \overline{\mathbb{R}} := \mathbb{R} \cup \{-\infty, \infty\}$ is called measurable (or Borel measurable) if $f^{-1}(A) \in \mathcal{F}$ for all open sets $A$ in $\mathbb{R}$.

Hence,

$$f^{-1}(\infty) = \bigcap_{n \in \mathbb{Z}} f^{-1}((n, \infty))$$

$$f^{-1}(-\infty) = \bigcap_{n \in \mathbb{Z}} f^{-1}((-\infty, n))$$

---

# Proposition

$f: \Omega \to \overline{\mathbb{R}}$ is measurable if and only if

1. $f^{-1}((a, \infty]) \in \mathcal{F} \quad \forall a \in \mathbb{R}$
2. $f^{-1}([-\infty, a)) \in \mathcal{F} \quad \forall a \in \mathbb{R}$
3. $f^-1 ((x,\infty]) \in \mathcal{F} \quad \forall a \in \mathbb{R}$
4. $f^-1 ((x,\infty)) \in \mathcal{F} \quad \forall a \in \mathbb{R}$
